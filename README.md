# emapta-server

1. Clone this repo 
```
[https://github.com/vlade2008/ListApp.git](https://github.com/vlade2008/emapta-server.git)
```
2. cd into directory and install the dependencies
```
cd emapta-server
npm install
```
3. Run server
```
npm run start
```
4. Fetch referrals
```
http://localhost:3000/referrals
```
